Open questions

  • What does this mean for the configurational case view?

Three types of case

Proposal

  • The central theoretical claim is that case comes in three types:
    • Semantic case
    • Lexical/Idiosyncractic case
    • Functional case
  • These are in competition with each other.

Semantic Case

Semantic Case

  • Not discussed much.
  • Semantic case is assigned to semantically restricted functions.

    1. Strákurinn
    2. the-boy
    1. beið
    2. waited
    1. allan
    2. all.acc
    1. daginn
    2. day.acc

'The boy waited all day.'

Idiosyncratic case vs. Functional case

Idiosyncratic case

  • "Associated with a particular thematic role."
  • "This type of case is assigned before thematic roles are associated with grammatical functions."

Functional case

  • "What is widely referred to as regular or default case [...] sensitive to surface relations."
  • There is a default system of case assignment that comes after other types of case assignment have already happened.

Association Principles

  • Linking of θ-roles:
    • If there is only one: assign to subject
    • If two: assign to subject and object
    • If three: assign to subject, object and 2object.
  • Agents are linked to subject
  • Default case marking: the highest available GF is assigned nominative, the next highest accusative.
